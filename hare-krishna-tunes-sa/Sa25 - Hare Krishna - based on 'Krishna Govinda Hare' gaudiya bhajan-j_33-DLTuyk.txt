========== Part 1 ==========

Starting from C or D, Govinda Prabhu plays both ways... If the previous ends with 'D', it seems to be better to start with 'D', otherwise 'C' is fine.

 DD#  D  C   Bb,  A,Bb, C
hare kṛṣṇa   hare kṛṣ  ṇa

      D       C  D      G  F D# DD#F
kṛṣṇa   kṛṣṇa   hare ha   re

 DD#  D C Bb,   A,Bb, C
hare rāma hare rā    ma

     D      C  D      G  F D# DD#F
rāma   rāma   hare ha   re
